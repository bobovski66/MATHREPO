Week 9 Summary: t-Distribution & Hypothesis Tests with z and t

What it is, when to use it, and how to test.

1. Why the t-distribution shows up

Sampling variability of s

When we estimate a population mean \(\mu\) using a sample mean \(\bar X\), the natural standardizer is \(\sigma / \sqrt{n}\). If \(\sigma\) is unknown, we plug in the sample SD \(s\), which itself varies. That extra uncertainty makes the standardized statistic

\[ T = \frac{\bar X - \mu}{s / \sqrt{n}} \]

follow a t-distribution (if data are normal), with degrees of freedom \(\text{df} = n - 1\).

Shape and behavior

  • The t-distribution is bell-shaped and centered at 0 like \(N(0,1)\), but has heavier tails.
  • As \(\text{df} \uparrow\), t looks more and more like \(N(0, 1)\).
  • Heavier tails \(\Rightarrow\) larger critical values than z for small/medium \(n\).

2. z vs. t: which one?

  • Use z when the population SD \(\sigma\) is known (or you have a very large sample with a well-justified process \(\sigma\)). Test statistic:

    \[ z = \frac{\bar x - \mu_0}{\sigma / \sqrt{n}}. \]

  • Use t when \(\sigma\) is unknown and estimated by \(s\) (the usual case). Test statistic:

    \[ t = \frac{\bar x - \mu_0}{s / \sqrt{n}}, \quad \text{df} = n - 1. \]

  • Assumptions (one-sample mean):
    • Random sample; independence.
    • For t: data are approximately normal, or \(n\) is moderate/large with no severe outliers.

3. Hypothesis testing: general flow (one-sample mean)

  1. State \(H_0: \mu = \mu_0\) and \(H_a\) (\(>\), <, or \(\neq\)).
  2. Choose the model: z (known \(\sigma\)) or t (unknown \(\sigma\)).
  3. Compute the test statistic (z or t) and the p-value.
  4. Decide: If \(p \le \alpha\), reject \(H_0\); else fail to reject.
  5. Conclude in context (plain language, mention direction if one-tailed).

4. Worked Example (using z; known \(\sigma\))

Set-up

A manufacturer claims mean drift \(\mu_0 = 50\). A sample of size \(n = 64\) gives \(\bar x = 53.1\). Historical process SD is known: \(\sigma = 12\). Test \(H_a: \mu > 50\) at \(\alpha = 0.05\).

Test

\[ z = \frac{53.1 - 50}{12 / \sqrt{64}} = \frac{3.1}{1.5} \approx 2.07. \]

This is a right-tailed test, so \[ p = P(Z \ge 2.07) \approx 0.019. \]

Decision

\(p \approx 0.019 < 0.05 \Rightarrow\) Reject \(H_0\).

Conclusion

There is evidence that the mean drift exceeds 50 units.

5. Worked Example (using t; unknown \(\sigma\))

Set-up

We test whether mean oxygen uptake differs from \(\mu_0 = 72\). Sample: \(n = 16\), \(\bar x = 74.5\), \(s = 5.6\). Two-sided test, \(\alpha = 0.05\).

Test

\[ t = \frac{74.5 - 72}{5.6 / \sqrt{16}} = \frac{2.5}{1.4} \approx 1.79,\quad \text{df} = 15. \]

Two-sided critical value at \(\alpha = 0.05\) with df = 15: \[ t^* \approx 2.131. \]

Decision

\(|t| = 1.79 < 2.131 \Rightarrow\) Fail to reject \(H_0\).

Conclusion

The data do not provide strong evidence that the mean differs from 72 (at 5%).

6. Formulas & quick reference

Test statistics (one-sample mean)

\[ z = \frac{\bar x - \mu_0}{\sigma / \sqrt{n}} \quad \text{(known \(\sigma\))} \] \[ t = \frac{\bar x - \mu_0}{s / \sqrt{n}}, \quad \text{df} = n - 1 \quad \text{(unknown \(\sigma\))} \]

p-values

  • Right-tailed: \(p = P(\text{stat} \ge \text{observed})\).
  • Left-tailed: \(p = P(\text{stat} \le \text{observed})\).
  • Two-tailed: \(p = 2P(\text{stat} \ge |\text{observed}|)\).

7. Common pitfalls (and fixes)

  • Using z when \(\sigma\) is unknown: If you don’t genuinely know \(\sigma\), use t with \(s\).
  • Two-tailed vs one-tailed: Match \(H_a\) to the question. Two-tailed doubles the tail area.
  • Mixing SD and SE: For testing \(\mu\), \(SE = \sigma / \sqrt{n}\) (or \(s / \sqrt{n}\)).
  • Normality for t: For small \(n\), check that data aren’t wildly skewed or driven by outliers.
  • Context in the conclusion: Always translate “reject / fail to reject” back to the story asked.
