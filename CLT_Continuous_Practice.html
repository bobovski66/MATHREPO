Central Limit Theorem: Full Solutions

1. Time to Walk to Campus (Normal Distribution)

Given: μ = 14, σ = 3, X = 18. We use the normal distribution:

Calculate z-score:

z = (18 - 14) / 3 = 1.33

Using a z-table or calculator: P(Z < 1.33) ≈ 0.9082

So, P(X > 18) = 1 - 0.9082 = 0.0918

2. Uniform Distribution, Single Sample (Flour Bags)

Uniform from a = 980 to b = 1020. Find P(X > 1010)

P = (1020 - 1010) / (1020 - 980) = 10 / 40 = 0.25

Answer: 0.25

3. Uniform Distribution, Sampling Distribution (36 Orders)

Uniform a = 10, b = 40. So:

  • μ = (10 + 40)/2 = 25
  • σ = (40 - 10)/√12 ≈ 8.66
  • σX̄ = σ / √n = 8.66 / √36 = 1.44

Find P(X̄ < 27):

z = (27 - 25)/1.44 ≈ 1.39

P(Z < 1.39) ≈ 0.9177

Answer: ≈ 0.9177

4. Normal Temperature Range

μ = 98.6, σ = 0.7. Find P(98 ≤ X ≤ 99):

  • z1 = (98 - 98.6)/0.7 ≈ -0.86
  • z2 = (99 - 98.6)/0.7 ≈ 0.57

Using z-table:

P(Z < 0.57) ≈ 0.7157, P(Z < -0.86) ≈ 0.1949

P = 0.7157 - 0.1949 = 0.5208

5. Skewed Distribution, Sampling Mean

μ = 50, σ = 5, n = 64, X̄ = 51.

σX̄ = 5 / √64 = 0.625

z = (51 - 50)/0.625 = 1.6

P(Z > 1.6) = 1 - P(Z < 1.6) ≈ 1 - 0.9452 = 0.0548
