Discrete Probability Distribution and Expected Value

A discrete probability distribution is a table or function that lists all possible values of a random variable along with their corresponding probabilities. The probabilities must sum to 1.

The expected value of a random variable, denoted by μ, represents the long-term average if the experiment is repeated many times. Itâ€™s calculated by summing the products of each possible value and its probability.
